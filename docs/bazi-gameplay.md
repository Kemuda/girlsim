# 八字 Gameplay 机制设计

> 命是给定的，运是选择的。命 + 运 = 命运。
> 这个游戏就是在玩「命运」这个词。

---

## 核心概念

八字系统的哲学基础：**你的出生决定了起点（命），但你活在什么时代、做什么选择，决定了走向（运）。**

这和 GirlSim 现有设计天然对齐：
- 角色面板（起点）= 命
- 玩家选择 = 运
- 影子代价 = 逆命的代价

八字版本把这个结构做得更显性：不是选一个「赫敏」，而是抽到一个命格，然后在一个时代里活出来。

---

## 游戏流程

```
① 抽时代（环境）→ ② 抽八字（命格）→ ③ 在格局里做选择 → ④ 结局
```

### ① 抽时代

你不选择自己出生在哪个时代。系统随机分配。

| 时代 | 特征 | 对游戏的影响 |
|------|------|------------|
| 古代（唐/宋/明） | 礼教森严，女性空间极小 | 志和渡的选项被压缩，影的积累更快 |
| 民国 | 新旧交替，缝隙中的自由 | 选项多但风险高，韧的考验密集 |
| 文革 | 政治吞噬一切 | 幽是危险的（内在世界被视为叛逆），身和志被强制征用 |
| 当代中国 | 选择爆炸，但内卷 | 所有维度都能发展，但时间/精力有限，选了一个就牺牲另一个 |
| 当代西方 | 自由度高，但孤独 | 志容易发展，渡反而更难（个体主义的代价） |
| 近未来 | 科技改变身体和社交 | 身的定义变了，幽可以被增强，但影也更隐蔽 |

时代不是装饰——它决定了**哪些选择存在、哪些选择不存在。**

古代女性没有「选择高风险事业」这个选项。这不是游戏限制，这就是现实。

### ② 抽八字

玩家抽到一个八字命格。系统按真实八字规则生成四柱，然后翻译成游戏面板。详细规则见下方「八字规则详解」。

### ③ 大运 × 人生阶段

八字系统里有「大运」概念——每 10 年换一个主导元素，影响你那个阶段的运势。

这和 GirlSim 的 7 个人生阶段天然对应。每个大运带来一个主导十神，决定这个阶段**出现什么类型的事件和选择**。

身强身弱决定的不是事件本身，而是事件对你的影响方向。同一个十神，在身强和身弱的盘里意味着完全不同的东西（详见下方规则）。

### ④ 结局：你活成了谁？

结局不再是通用的维度结局，而是匹配到**角色原型的变体**：

```
你的八字：金金水木（高志高幽）
你的时代：当代中国
你的选择：持续投资渡，压低了志
最终面板：志 2  幽 5  韧 3  渡 5  身 2  影 3

→ "你活成了一个没有出家的李清照。
   你有她的深沉和才华，但你选择了把它传递出去，
   而不是独自承受。"
```

17 个角色成为**结局的词汇表**——不是「你是哪个角色」，而是「你最像谁的哪个版本」。

## 八字规则详解

### 四柱结构

八字 = 四柱，每柱有天干 + 地支：

```
        年柱      月柱      日柱      时柱
天干    甲(木)    丙(火)    壬(水)    庚(金)     ← 外在：别人怎么看你
地支    子(水)    寅(木)    午(火)    申(金)     ← 内在：你实际是什么
                            ↑
                          日主（元女）
                    你的坐标原点，一切从这里算
```

- **天干**（外在）→ 对应「主星」，代表一个人外显的、别人看到的特质
- **地支**（内在）→ 藏有「藏干」，对应「副星」，代表内在的、实质的东西

### 日主：你的坐标原点

日柱的天干 = 你的**元女**（女性角色所以叫元女）。它的五行决定了你是一个什么「盘」。

所有其他干支的五行，都**相对于日主**来定义含义：

```
以日主为「水」为例：

水 → 和我同类   → 比肩 / 劫财
木 → 我生的     → 食神 / 伤官
火 → 我克的     → 偏财 / 正财
土 → 克我的     → 七杀 / 正官
金 → 生我的     → 偏印 / 正印
```

同类五行，阴阳一致 = 偏，阴阳不一致 = 正。

**日主不同，同一个五行的含义完全不同。** 火对水盘来说是「财」，对金盘来说是「官杀」。这就是八字的坐标系。

### 十神：人生事件的分类

十神不是抽象概念——它们是人生中会遇到的**事件类型**。

| 十神 | 五行关系 | 含义 | 游戏中的事件/选择 |
|------|---------|------|-----------------|
| **比肩** | 同我·同阴阳 | 平等的同行者 | 遇到志同道合的人。合作还是各走各路？ |
| **劫财** | 同我·异阴阳 | 争夺者 | 你的成果被分走，或你分走别人的（取决于身强弱） |
| **食神** | 我生·同阴阳 | 温柔的表达 | 创作、教育、养育。把内在的东西温和地输出 |
| **伤官** | 我生·异阴阳 | 叛逆的表达 | 打破规则、激烈创新。有代价，但可能改变世界 |
| **偏财** | 我克·同阴阳 | 流动的财 | 冒险、投机、意外机遇。来得快去得也快 |
| **正财** | 我克·异阴阳 | 稳定的财 | 稳定工作、积累、务实的人生选择 |
| **七杀** | 克我·同阴阳 | 强烈的压力 | 危机、强敌、逼你成长的力量。不死就变强 |
| **正官** | 克我·异阴阳 | 正当的权威 | 规则、体制、社会期待、上司。你服从还是反抗？ |
| **偏印** | 生我·同阴阳 | 偏门的滋养 | 非主流的灵感、孤独中的顿悟、野路子的智慧 |
| **正印** | 生我·异阴阳 | 正统的滋养 | 母亲、学校、传统教育、被体制认可的支持 |

### 月柱地支：你的人生主题

月柱的地支藏干所对应的十神，决定了这个人**一辈子在跟什么较劲**（what her life is about）。

| 月柱藏干对应的十神 | 人生主基调 | 游戏中的体现 |
|------------------|-----------|------------|
| 正财 | 一辈子在追求稳定和积累 | 正财相关的选择反复出现，是她的主旋律 |
| 七杀 | 一辈子在跟压力和危机搏斗 | 每个阶段都有七杀事件，逃不掉 |
| 伤官 | 一辈子在叛逆和表达中挣扎 | 不断面对「顺从还是打破」的选择 |
| 正印 | 一辈子被体制和传统滋养/束缚 | 母亲、学校、权威机构反复出现 |
| 食神 | 一辈子在温和地创造和输出 | 创作、教育、养育是贯穿的线索 |
| ... | ... | ... |

月柱主题不会改变——它是命的一部分。你可以选择怎么回应它，但你无法选择不面对它。

### 身强身弱：同一张牌，两种意思

数命局中日主元素得到多少支持（同类五行 + 生我五行的数量和位置），判断身强还是身弱。

**这是八字系统中最关键的游戏机制：同一个十神事件，在身强和身弱的盘里意味着相反的东西。**

| 十神 | 身强时 | 身弱时 |
|------|--------|--------|
| **劫财** | 别人抢你的功劳和资源 | 你反而能借到别人的力量 |
| **七杀** | 一个值得的对手，让你更强 | 压倒性的压力，可能摧毁你 |
| **正财** | 你能稳稳抓住机会 | 机会在眼前但你没力气够到 |
| **伤官** | 你有足够的能量去叛逆 | 叛逆会耗尽你，得不偿失 |
| **正印** | 你不需要这个支持，它反而是束缚 | 这是你急需的滋养和保护 |

**游戏中的体现：**

同一个场景——比如「公司要裁员，你是被裁的候选人之一」（七杀事件）：

- **身强的玩家**看到的选项可能包括「主动请缨负责最难的项目」「直接跟老板摊牌」
- **身弱的玩家**看到的选项可能是「默默加班证明自己」「找人帮忙说情」「考虑离开」

不是不同的故事，是同一个压力下**不同的应对资源**。

### 透出

当一个地支的藏干（内在的、隐藏的特质）在天干中也出现了同一个五行，这个特质就**透出**了——从内在变成了外在可见的。

```
例：地支藏干有火（内心有渡/联结的渴望），天干也出现火
→ 这个火「透出」了
→ 她的联结不是隐藏的愿望，而是所有人都能看到的特质
```

**游戏中的体现：** 透出的十神会成为**这个角色最鲜明的标签**——别人第一眼就能看到的东西。未透出的藏干则是她自己知道但别人不一定看到的暗线。

### 五行 → 六维映射

| 五行 | 维度 | 对应关系 |
|------|------|---------|
| 金 | 志（Coherence） | 金主决断、方向、锋利 |
| 水 | 幽（Depth） | 水主深沉、流动、隐藏 |
| 木 | 韧（Regeneration） | 木主生长、柔韧、再生 |
| 火 | 渡（Transmission） | 火主传播、温暖、联结 |
| 土 | 身（Body） | 土主承载、扎根、稳定 |

**影（Shadow）= 命格的偏枯程度 + 未透出的藏干。**

- 五行越不平衡，起始影越高——太多东西被压抑了
- 未透出的藏干也贡献影——你内在有的东西，但世界看不到

### 面板生成规则

系统按真实八字规则生成四柱（年月日时 × 天干地支），然后：

1. **确定日主**（日柱天干的五行）→ 建立坐标系
2. **算十神**：其余七个干支各自对应一个十神
3. **算身强弱**：数日主得到的支持（比肩、劫财、正印、偏印的数量和位置）
4. **定月柱主题**：月柱地支藏干对应的十神 = 人生主基调
5. **查透出**：哪些藏干在天干中出现了
6. **生成六维面板**：五行分布 → 六维初始值；偏枯度 + 未透出数 → 影值

```
例：日主壬水（阳水），命局：金×2 水×2 木×1 火×1 土×1 （略偏金水）
→ 身偏强（水得金生，又有比肩）
→ 志 4  幽 4  韧 2  渡 2  身 2  影 3
→ 月柱藏干对应「伤官」→ 一辈子在叛逆和创造中挣扎
→ 伤官透出 → 这种叛逆是所有人都看得到的
```

### 大运 × 人生阶段

每个人生阶段的大运带来一个主导十神，决定这个阶段出现什么类型的事件。

大运的十神也是相对于日主来算的——同一个大运元素，对不同日主的玩家意味着不同的十神。

| 大运十神 | 这个阶段的基调 | 身强时 | 身弱时 |
|---------|--------------|--------|--------|
| 比肩运 | 同行者涌现 | 竞争加剧，但实力在 | 终于找到同类，不再孤独 |
| 七杀运 | 压力和危机 | 淬炼期，扛过去就飞跃 | 至暗时刻，需要外援或退让 |
| 食神运 | 创作和表达 | 丰收期，作品涌现 | 小确幸，温柔但力量有限 |
| 正财运 | 稳定和积累 | 财来得容易，但可能无聊 | 终于有了安全感 |
| 偏印运 | 灵感和孤独 | 走入偏门，可能走火入魔 | 在黑暗中摸到一束光 |
| ... | ... | ... | ... |

**相生相克自动产生人生节奏：** 不需要手动编排哪个阶段是高潮/低谷，十神的流转天然创造起伏。

## 与现有系统的关系

| 现有设计 | 八字版本 | 关系 |
|---------|---------|------|
| 选角色（赫敏/三毛） | 抽八字命格 | 替代：从「选一个人」变成「抽一个命」 |
| 固定初始面板 | 五行生成面板 | 替代：随机性更强，每局不同 |
| 7 个人生阶段 | 大运 × 7 阶段 | 增强：阶段自带元素加成/克制 |
| 影子代价 | 逆命代价 | 相同概念，更强的文化表达 |
| 叙事感知系统 | 不变 | 保留 |
| 通用结局文本 | 角色原型匹配 | 替代：结局更个人化 |
| — | 时代系统 | 新增：环境层 |

**不是推翻重来，而是在现有骨架上换一层皮肤。** 六维系统、场景结构、影子代价、叙事感知——这些都保留。八字提供的是一套更有文化共鸣的「入口」和「出口」。

## 开放问题

- [ ] 时代系统的颗粒度：是大的时代（古代/当代），还是具体到十年（1970s/1990s/2010s）？
- [ ] 身强身弱的判定要多精确？简化版（数五行个数）还是考虑十二长生、根气等？
- [ ] 十神对六维的影响是固定映射，还是也受身强弱调节？
- [ ] 透出的游戏表现：是UI上直接显示「你的伤官透出了」，还是隐性地体现在叙事和选项里？
- [ ] 月柱主题是显性告知玩家（"你的人生主题是：伤官"），还是让玩家自己在反复出现的事件中领悟？
- [ ] Demo 日策略：让观众输入真实生辰 → 生成八字 → 开始玩？这个入口够不够 wow？
- [ ] *note to self*：「透出」的完整定义需要再确认——藏干在天干出现 = 透出，但具体的判定规则（是否需要同柱、是否有距离限制）需要查证

**你的命是抽的，你的时代是抽的，但你的选择是你的。这就是命运。**
