# 八字 Gameplay 机制设计

> 命是给定的，运是选择的。命 + 运 = 命运。
> 这个游戏就是在玩「命运」这个词。

---

## 核心概念

八字系统的哲学基础：**你的出生决定了起点（命），但你活在什么时代、做什么选择，决定了走向（运）。**

这和 GirlSim 现有设计天然对齐：
- 角色面板（起点）= 命
- 玩家选择 = 运
- 影子代价 = 逆命的代价

八字版本把这个结构做得更显性：不是选一个「赫敏」，而是抽到一个命格，然后在一个时代里活出来。

---

## 游戏流程

```
① 抽时代（环境）→ ② 抽八字（命格）→ ③ 在格局里做选择 → ④ 结局
```

### ① 抽时代

你不选择自己出生在哪个时代。系统随机分配。

| 时代 | 特征 | 对游戏的影响 |
|------|------|------------|
| 古代（唐/宋/明） | 礼教森严，女性空间极小 | 志和渡的选项被压缩，影的积累更快 |
| 民国 | 新旧交替，缝隙中的自由 | 选项多但风险高，韧的考验密集 |
| 文革 | 政治吞噬一切 | 幽是危险的（内在世界被视为叛逆），身和志被强制征用 |
| 当代中国 | 选择爆炸，但内卷 | 所有维度都能发展，但时间/精力有限，选了一个就牺牲另一个 |
| 当代西方 | 自由度高，但孤独 | 志容易发展，渡反而更难（个体主义的代价） |
| 近未来 | 科技改变身体和社交 | 身的定义变了，幽可以被增强，但影也更隐蔽 |

时代不是装饰——它决定了**哪些选择存在、哪些选择不存在。**

古代女性没有「选择高风险事业」这个选项。这不是游戏限制，这就是现实。

### ② 抽八字

玩家抽到一个简化的八字命格。不需要懂命理——系统直接翻译成六维面板。

**五行 → 六维映射：**

| 五行 | 维度 | 对应关系 |
|------|------|---------|
| 金 | 志（Coherence） | 金主决断、方向、锋利 |
| 水 | 幽（Depth） | 水主深沉、流动、隐藏 |
| 木 | 韧（Regeneration） | 木主生长、柔韧、再生 |
| 火 | 渡（Transmission） | 火主传播、温暖、联结 |
| 土 | 身（Body） | 土主承载、扎根、稳定 |

**影（Shadow）= 命格的偏枯程度。**

八字里五行越不平衡，起始影值越高。一个全是金水的命格（极端的志+幽），影天然就高——因为她有太多东西被压抑了。

**生成规则（简化版）：**

系统随机生成 4 柱，每柱带一个主元素：

```
例：金 金 水 木
→ 金×2 水×1 木×1 火×0 土×0
→ 志 4  幽 3  韧 2  渡 1  身 1  影 4（严重偏枯）
```

```
例：火 土 木 水
→ 各 ×1，金 ×0
→ 志 1  幽 3  韧 3  渡 3  身 3  影 2（较均衡，但志缺失）
```

缺失的元素 = 人生中需要补的功课。影的高低 = 内在张力的大小。

### ③ 大运 × 人生阶段

八字系统里有「大运」概念——每 10 年换一个主导元素，影响你那个阶段的运势。

这和 GirlSim 的 7 个人生阶段天然对应：

| 人生阶段 | 大运元素（随机） | 效果 |
|---------|---------------|------|
| Girlhood (0-12) | 木 | 木生火 → 这个阶段渡（联结）的选项更丰富 |
| The Threshold (12-18) | 火 | 火克金 → 志（方向）受到挑战 |
| First World (18-28) | 土 | 土生金 → 志的发展窗口打开 |
| ... | ... | ... |

**相生 vs 相克：**

- 大运元素**生**你的强项 → 顺风期，好选项多
- 大运元素**克**你的强项 → 逆风期，需要更多韧
- 大运元素**补**你的缺失 → 成长期，机会来了但要抓住

这创造了自然的人生节奏——不需要手动编排「第三阶段是困难的」，五行相生相克自动产生起伏。

### ④ 结局：你活成了谁？

结局不再是通用的维度结局，而是匹配到**角色原型的变体**：

```
你的八字：金金水木（高志高幽）
你的时代：当代中国
你的选择：持续投资渡，压低了志
最终面板：志 2  幽 5  韧 3  渡 5  身 2  影 3

→ "你活成了一个没有出家的李清照。
   你有她的深沉和才华，但你选择了把它传递出去，
   而不是独自承受。"
```

17 个角色成为**结局的词汇表**——不是「你是哪个角色」，而是「你最像谁的哪个版本」。

## 与现有系统的关系

| 现有设计 | 八字版本 | 关系 |
|---------|---------|------|
| 选角色（赫敏/三毛） | 抽八字命格 | 替代：从「选一个人」变成「抽一个命」 |
| 固定初始面板 | 五行生成面板 | 替代：随机性更强，每局不同 |
| 7 个人生阶段 | 大运 × 7 阶段 | 增强：阶段自带元素加成/克制 |
| 影子代价 | 逆命代价 | 相同概念，更强的文化表达 |
| 叙事感知系统 | 不变 | 保留 |
| 通用结局文本 | 角色原型匹配 | 替代：结局更个人化 |
| — | 时代系统 | 新增：环境层 |

**不是推翻重来，而是在现有骨架上换一层皮肤。** 六维系统、场景结构、影子代价、叙事感知——这些都保留。八字提供的是一套更有文化共鸣的「入口」和「出口」。

## 开放问题

- [ ] 时代系统的颗粒度：是大的时代（古代/当代），还是具体到十年（1970s/1990s/2010s）？
- [ ] 八字简化到什么程度？4 柱 × 五行已经足够，还是需要天干地支的层次？
- [ ] 相生相克的计算是显性的（告诉玩家「这是顺风期」）还是隐性的（只通过选项丰富度体现）？
- [ ] Demo 日策略：让观众输入真实生辰 → 生成八字 → 开始玩？这个入口够不够 wow？
