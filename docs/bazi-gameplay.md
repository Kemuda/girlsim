# 八字 Gameplay 机制设计

> 命是给定的，运是选择的。命 + 运 = 命运。
> 这个游戏就是在玩「命运」这个词。

这是一个**基于八字命理体系的人生模拟游戏**的机制设计文档。

八字本身就是完整的状态系统——五行消长是状态，十神是事件语言，偏枯是隐患。游戏的视觉界面需要让不懂八字的人也能感受到五行的流动——环形图、相生相克箭头、透出元素发光等。术语在后台，感受在前台。

---

## 核心概念

八字系统的哲学基础：**你的出生决定了起点（命），但你活在什么时代、做什么选择，决定了走向（运）。**

玩家抽到一个命格，被分配到一个时代，然后在这个格局里做选择——这就是在玩「命运」这个词。

---

## 游戏流程

```
① 抽时代（环境）→ ② 抽八字（命格）→ ③ 在格局里做选择 → ④ 结局
```

### ① 抽时代

你不选择自己出生在哪个时代。系统随机分配。

| 时代 | 特征 | 对游戏的影响 |
|------|------|------------|
| 古代（唐/宋/明） | 礼教森严，女性空间极小 | 志和渡的选项被压缩，影的积累更快 |
| 民国 | 新旧交替，缝隙中的自由 | 选项多但风险高，韧的考验密集 |
| 文革 | 政治吞噬一切 | 幽是危险的（内在世界被视为叛逆），身和志被强制征用 |
| 当代中国 | 选择爆炸，但内卷 | 所有维度都能发展，但时间/精力有限，选了一个就牺牲另一个 |
| 当代西方 | 自由度高，但孤独 | 志容易发展，渡反而更难（个体主义的代价） |
| 近未来 | 科技改变身体和社交 | 身的定义变了，幽可以被增强，但影也更隐蔽 |

时代不是装饰——它决定了**哪些选择存在、哪些选择不存在。**

古代女性没有「选择高风险事业」这个选项。这不是游戏限制，这就是现实。

### ② 抽八字

玩家抽到一个八字命格。系统按真实八字规则生成四柱，然后翻译成游戏面板。详细规则见下方「八字规则详解」。

### ③ 大运 × 人生阶段

八字系统里有「大运」概念——每 10 年换一个主导元素，影响你那个阶段的运势。

这和 GirlSim 的 7 个人生阶段天然对应。每个大运带来一个主导十神，决定这个阶段**出现什么类型的事件和选择**。

身强身弱决定的不是事件本身，而是事件对你的影响方向。同一个十神，在身强和身弱的盘里意味着完全不同的东西（详见下方规则）。

### ④ 结局：你活成了谁？

结局不再是通用的维度结局，而是匹配到**角色原型的变体**：

```
你的八字：金金水木（高志高幽）
你的时代：当代中国
你的选择：持续投资渡，压低了志
最终面板：志 2  幽 5  韧 3  渡 5  身 2  影 3

→ "你活成了一个没有出家的李清照。
   你有她的深沉和才华，但你选择了把它传递出去，
   而不是独自承受。"
```

17 个角色成为**结局的词汇表**——不是「你是哪个角色」，而是「你最像谁的哪个版本」。

## 八字规则详解

### 四柱结构

八字 = 四柱，每柱有天干 + 地支：

```
        年柱      月柱      日柱      时柱
天干    甲(木)    丙(火)    壬(水)    庚(金)     ← 外在：别人怎么看你
地支    子(水)    寅(木)    午(火)    申(金)     ← 内在：你实际是什么
                            ↑
                          日主（元女）
                    你的坐标原点，一切从这里算
```

- **天干**（外在）→ 对应「主星」，代表一个人外显的、别人看到的特质
- **地支**（内在）→ 藏有「藏干」，对应「副星」，代表内在的、实质的东西

### 日主：你的坐标原点

日柱的天干 = 你的**元女**（女性角色所以叫元女）。它的五行决定了你是一个什么「盘」。

所有其他干支的五行，都**相对于日主**来定义含义：

```
以日主为「水」为例：

水 → 和我同类   → 比肩 / 劫财
木 → 我生的     → 食神 / 伤官
火 → 我克的     → 偏财 / 正财
土 → 克我的     → 七杀 / 正官
金 → 生我的     → 偏印 / 正印
```

同类五行，阴阳一致 = 偏，阴阳不一致 = 正。

**日主不同，同一个五行的含义完全不同。** 火对水盘来说是「财」，对金盘来说是「官杀」。这就是八字的坐标系。

### 十神：人生事件的分类

十神不是抽象概念——它们是人生中会遇到的**事件类型**。

| 十神 | 五行关系 | 含义 | 游戏中的事件/选择 |
|------|---------|------|-----------------|
| **比肩** | 同我·同阴阳 | 平等的同行者 | 遇到志同道合的人。合作还是各走各路？ |
| **劫财** | 同我·异阴阳 | 争夺者 | 你的成果被分走，或你分走别人的（取决于身强弱） |
| **食神** | 我生·同阴阳 | 温柔的表达 | 创作、教育、养育。把内在的东西温和地输出 |
| **伤官** | 我生·异阴阳 | 叛逆的表达 | 打破规则、激烈创新。有代价，但可能改变世界 |
| **偏财** | 我克·同阴阳 | 流动的财 | 冒险、投机、意外机遇。来得快去得也快 |
| **正财** | 我克·异阴阳 | 稳定的财 | 稳定工作、积累、务实的人生选择 |
| **七杀** | 克我·同阴阳 | 强烈的压力 | 危机、强敌、逼你成长的力量。不死就变强 |
| **正官** | 克我·异阴阳 | 正当的权威 | 规则、体制、社会期待、上司。你服从还是反抗？ |
| **偏印** | 生我·同阴阳 | 偏门的滋养 | 非主流的灵感、孤独中的顿悟、野路子的智慧 |
| **正印** | 生我·异阴阳 | 正统的滋养 | 母亲、学校、传统教育、被体制认可的支持 |

### 月柱地支：你的人生主题

月柱的地支藏干所对应的十神，决定了这个人**一辈子在跟什么较劲**（what her life is about）。

| 月柱藏干对应的十神 | 人生主基调 | 游戏中的体现 |
|------------------|-----------|------------|
| **比肩** | 一辈子在「我是谁」的身份认同中打转 | 不断遇到和自己相似的人——是同行者还是镜子？合作还是被取代？核心问题：独立 vs 合群 |
| **劫财** | 一辈子在争夺和被争夺中度过 | 身强：你总是在赢，但赢来的东西总被人分走。身弱：你总是借着别人的势往上爬。核心问题：什么是真正属于你的？ |
| **食神** | 一辈子在温和地创造和输出 | 创作、教育、养育是贯穿的线索。她的满足感来自「我做出了一个好东西」。核心问题：表达自己 vs 照顾别人 |
| **伤官** | 一辈子在叛逆和表达中挣扎 | 不断面对「顺从还是打破」的选择。她的才华是锋利的，容易伤人也伤己。核心问题：要磨平棱角还是刺穿世界？ |
| **偏财** | 一辈子在冒险和流动中寻找意义 | 机会多但不稳定，来得快去得快。她的人生节奏是一波接一波。核心问题：要不要赌这一把？ |
| **正财** | 一辈子在追求稳定和积累 | 正财相关的选择反复出现。她的安全感来自「我拥有、我控制」。核心问题：稳定是保护还是牢笼？ |
| **七杀** | 一辈子在跟压力和危机搏斗 | 每个阶段都有七杀事件，逃不掉。她活在高压下，要么被锻造要么被碾碎。核心问题：对抗还是臣服？ |
| **正官** | 一辈子在规则和期待中寻找自我 | 体制、权威、社会角色反复定义她。她被看见，但看见的是真的她吗？核心问题：满足期待 vs 做自己 |
| **偏印** | 一辈子在孤独和灵感之间游走 | 她的智慧来自非主流的地方——直觉、异类经历、边缘视角。核心问题：孤独是诅咒还是天赋？ |
| **正印** | 一辈子被体制和传统滋养/束缚 | 母亲、学校、权威机构反复出现。她被保护得很好，但也被框住了。核心问题：感恩还是挣脱？ |

每个玩家的月柱主题不会改变——它是命的一部分。你可以选择怎么回应它，但你无法选择不面对它。

**游戏中的实现：** 无论大运带来什么十神事件，月柱主题的相关事件**每个阶段都会出现至少一次**。如果你的月柱是七杀，那不管这个阶段的大运是食神运还是正财运，你都会遇到一次压力事件。这是你的宿命回响。

### 身强身弱：同一张牌，两种意思

身强身弱不是数个数，是看**能量在这个盘里怎么流动**。一个盘里有三个火不代表火就强——要看每个火在什么位置、旁边是什么、整个环境支不支持它。

#### 四个维度判断身强弱

传统上叫「得令、得地、得生、得助」：

**1. 得令 — 你出生在什么季节？（权重最大）**

月柱地支 = 月令 = 整个盘的大气候。这是判断身强弱**权重最大的单一因素**。

```
丁火日主：

月令是巳/午（夏天）→ 火当令，天然身强
月令是亥/子（冬天）→ 火失令，天然身弱
月令是寅/卯（春天）→ 木当令，木生火，间接支持
月令是申/酉（秋天）→ 金当令，火不得令
```

就算盘里有三个火，如果月令是冬天（水旺），这三个火也可能都是弱的——因为整个环境不支持火。

**2. 得地 — 你坐在什么上面？**

日支（日柱地支）是日主的「座位」，离日主最近。看日主在日支的十二长生阶段：

```
丁火坐午 → 临官，坐在自己的旺地上，非常实
丁火坐亥 → 胎，坐在克自己的元素上，虚弱
丁火坐寅 → 有木生火，有人托着你
```

十二长生阶段越接近帝旺/临官，日主越强；越接近死/绝/胎，日主越弱。

**十个天干的长生起点（查表用）：**

| 天干 | 五行 | 阴阳 | 长生起于 | 方向 |
|------|------|------|---------|------|
| 甲 | 木 | 阳 | 亥 | 顺行 |
| 乙 | 木 | 阴 | 午 | 逆行 |
| 丙 | 火 | 阳 | 寅 | 顺行 |
| 丁 | 火 | 阴 | 酉 | 逆行 |
| 戊 | 土 | 阳 | 寅 | 顺行 |
| 己 | 土 | 阴 | 酉 | 逆行 |
| 庚 | 金 | 阳 | 巳 | 顺行 |
| 辛 | 金 | 阴 | 子 | 逆行 |
| 壬 | 水 | 阳 | 申 | 顺行 |
| 癸 | 水 | 阴 | 卯 | 逆行 |

十二长生周期：长生 → 沐浴 → 冠带 → 临官 → 帝旺 → 衰 → 病 → 死 → 墓 → 绝 → 胎 → 养

从起点按方向（顺/逆）走十二地支（子丑寅卯辰巳午未申酉戌亥），即可查出任何天干在任何地支上的阶段。

**3. 得生 — 有没有人喂你？**

盘里有没有生你的元素（对火来说是木），以及这个木**自身强不强、在什么位置**。

```
一个木在年柱 → 离日主远，影响小（像远方亲戚给你寄钱）
一个木在月柱或时柱 → 离日主近，直接喂你
一个木自己坐在死/绝上 → 它自己都快没了，喂不了你多少
一个木自己坐在帝旺上 → 它很旺，能源源不断生火
```

**4. 得助 — 有没有同类帮你？**

盘里其他的火在哪、强不强。

```
旁边有一个丙火坐寅 → 强有力的帮手
旁边有一个丁火坐子 → 也是弱火，两个弱的凑一起也强不到哪去
```

#### 位置 = 权重

**八字不是八个元素的平等投票，而是一个有距离、有方向的能量场。**

```
        年柱      月柱      日柱      时柱
                            ↑ 日主

离日主的距离：  远        近        自己       近

影响力权重：    小        大        —         中
              (祖辈)    (环境)    (自己)     (子女/晚年)
```

**相邻的柱之间直接互动：**

- 年支 ↔ 月支 → 直接互动
- 月支 ↔ 日支 → 直接互动
- 日支 ↔ 时支 → 直接互动
- 年支 ↔ 时支 → **不直接互动**（隔着两柱）

#### 地支之间的复杂互动

地支之间不只有五行相生相克，还有：

| 互动类型 | 含义 | 效果 | 例 |
|---------|------|------|-----|
| **六合** | 两个地支合在一起 | 可能化成另一种五行 | 子丑合土、寅亥合木 |
| **三合** | 三个地支合局 | 形成极强的某种五行 | 寅午戌合火局、申子辰合水局 |
| **相冲** | 两个对立地支互冲 | 双方都受损，能量剧烈碰撞 | 子午冲、卯酉冲 |
| **相刑** | 特定组合产生内部矛盾 | 自我消耗、内在冲突 | 寅巳申三刑 |

**对游戏的意义：** 合局可以改变五行力量的格局（三个本来不起眼的地支合在一起突然变成最强的力量），冲可以打破现有平衡。这些在大运流转时特别重要——新来的大运地支可能和命局里的地支形成合或冲，导致能量格局突变。

#### 根气

「根」= 天干在地支里有没有根基。

天干是露在外面的（别人看到的），地支是藏在下面的（内在的）。如果一个天干的五行在地支的藏干里也出现了，那这个天干就**有根**——有实质支撑。

```
例：日主壬水，地支里有「子」（藏干癸水）
→ 水有根 → 实的，有力量

例：日主壬水，四个地支里没有任何含水的藏干
→ 水无根 → 虚的，像没根的植物
```

「气」= 根的质量。根多气旺，根少气衰。

**对游戏的意义：** 有根 = 特质是实的，无根 = 特质是虚的。一个有根的伤官 = 真正的叛逆者，有底气去打破。一个无根的伤官 = 内心想叛逆但没有实力，可能只是嘴上说说。

#### 完整示例

```
        年柱      月柱      日柱      时柱
天干    甲(木)    壬(水)    丁(火)    庚(金)
地支    寅(木)    子(水)    亥(水)    申(金)
```

丁火日主，逐项分析：

1. **得令？** 月令子（冬天，水旺）→ ✗ 失令，大环境对火极不利
2. **得地？** 坐亥（水，十二长生在「胎」）→ ✗ 失地，坐在克自己的元素上
3. **得生？** 年柱甲木坐寅（木旺）→ △ 有人生火，但在年柱，离得远
4. **得助？** 盘里没有其他火 → ✗ 无助
5. **根气？** 四个地支（寅子亥申）的藏干里有没有火？寅藏丙火 → △ 有一点根，但在年支，远
6. **地支互动？** 亥子相邻且都是水，水势更强；申子辰如果凑齐可合水局（目前缺辰）

**结论：身极弱。** 虽然年柱甲木可以生火，但月令水旺、日坐亥水、时柱庚金又生水，整个盘的能量往水那边流。这个丁火是冬天里的一根蜡烛。

**这个人的游戏体验：** 内心有热情（毕竟是火），但环境一直在压她。她的人生主题可能是「在不利的环境中保护自己的那一点点火光」。如果大运带来木或火，那是她的翻身机会。如果大运是寅午戌（凑成火局），那是命运级的转折——小火突然变大火。

#### 身强弱是一个光谱

综合以上所有因素，身强弱不是二选一，而是一个连续光谱：

```
极弱 ← ─ ─ ─ 偏弱 ─ ─ ─ 中和 ─ ─ ─ 偏强 ─ ─ ─ → 极强
```

**游戏中的体现：** 同一个十神事件，不再只有「身强版」和「身弱版」两种反应，而是一个光谱上的多种反应。

**这是八字系统中最关键的游戏机制：同一个十神事件，在不同身强弱程度的盘里意味着不同的东西。**

| 十神 | 身强时 | 身弱时 |
|------|--------|--------|
| **劫财** | 别人抢你的功劳和资源 | 你反而能借到别人的力量 |
| **七杀** | 一个值得的对手，让你更强 | 压倒性的压力，可能摧毁你 |
| **正财** | 你能稳稳抓住机会 | 机会在眼前但你没力气够到 |
| **伤官** | 你有足够的能量去叛逆 | 叛逆会耗尽你，得不偿失 |
| **正印** | 你不需要这个支持，它反而是束缚 | 这是你急需的滋养和保护 |

**场景示例：**

「公司要裁员，你是被裁的候选人之一」（七杀事件）：

- **身强的玩家**看到的选项：「主动请缨负责最难的项目」「直接跟老板摊牌」
- **身弱的玩家**看到的选项：「默默加班证明自己」「找人帮忙说情」「考虑离开」

不是不同的故事，是同一个压力下**不同的应对资源**。

### 透出

当一个地支的藏干（内在的、隐藏的特质）在天干中也出现了同一个五行，这个特质就**透出**了——从内在变成了外在可见的。

```
例：地支藏干有火，天干也出现火
→ 这个火「透出」了
→ 她的火不是隐藏的渴望，而是所有人都能看到的特质
```

**游戏中的体现：** 透出的十神会成为**这个角色最鲜明的标签**——别人第一眼就能看到的东西。未透出的藏干则是她自己知道但别人不一定看到的暗线。

### 五行的性格语言

| 五行 | 性格特质 |
|------|---------|
| 金 | 决断、方向、锋利、肃杀 |
| 水 | 深沉、流动、隐藏、智慧 |
| 木 | 生长、柔韧、再生、仁慈 |
| 火 | 传播、温暖、联结、礼义 |
| 土 | 承载、扎根、稳定、信实 |

命格的偏枯（五行严重不平衡）= 内在张力的来源。越不平衡，这个人内在的矛盾越大——太多东西被压抑了。未透出的藏干也是隐患——你内在有的东西，但世界看不到。

### 面板生成规则

系统按真实八字规则生成四柱（年月日时 × 天干地支），然后：

1. **确定日主**（日柱天干的五行）→ 建立坐标系
2. **算十神**：其余七个干支各自对应一个十神
3. **算身强弱**：数日主得到的支持（比肩、劫财、正印、偏印的数量和位置）
4. **定月柱主题**：月柱地支藏干对应的十神 = 人生主基调
5. **查透出**：哪些藏干在天干中出现了
6. **生成六维面板**：五行分布 → 六维初始值；偏枯度 + 未透出数 → 影值

```
例：日主壬水（阳水），命局：金×2 水×2 木×1 火×1 土×1 （略偏金水）
→ 身偏强（水得金生，又有比肩）
→ 志 4  幽 4  韧 2  渡 2  身 2  影 3
→ 月柱藏干对应「伤官」→ 一辈子在叛逆和创造中挣扎
→ 伤官透出 → 这种叛逆是所有人都看得到的
```

### 大运 × 人生阶段

每个人生阶段的大运带来一个主导十神，决定这个阶段出现什么类型的事件。

大运的十神也是相对于日主来算的——同一个大运元素，对不同日主的玩家意味着不同的十神。

| 大运十神 | 这个阶段的基调 | 身强时 | 身弱时 |
|---------|--------------|--------|--------|
| 比肩运 | 同行者涌现 | 竞争加剧，但实力在 | 终于找到同类，不再孤独 |
| 七杀运 | 压力和危机 | 淬炼期，扛过去就飞跃 | 至暗时刻，需要外援或退让 |
| 食神运 | 创作和表达 | 丰收期，作品涌现 | 小确幸，温柔但力量有限 |
| 正财运 | 稳定和积累 | 财来得容易，但可能无聊 | 终于有了安全感 |
| 偏印运 | 灵感和孤独 | 走入偏门，可能走火入魔 | 在黑暗中摸到一束光 |
| ... | ... | ... | ... |

**相生相克自动产生人生节奏：** 不需要手动编排哪个阶段是高潮/低谷，十神的流转天然创造起伏。
