# Universe Map — 角色宇宙地图

> 每个角色是一个宇宙。每个宇宙有独特的六维起点和人生决策线。
> 编辑叙事内容时的核心参考文档。

---

## 六维

| 字 | 英文 | 含义 |
|----|------|------|
| **志** | Coherence | 心之所之——方向感与意志力 |
| **幽** | Depth | 隐而深——内在世界的丰富 |
| **韧** | Regeneration | 柔而固——跌倒后站起来的力量 |
| **渡** | Transmission | 济人渡己——对他人的影响与联结 |
| **身** | Body | 安身立命——扎根在肉身里的智慧 |
| **影** | Shadow | 有光才有影——被压抑的、未说出口的部分 |

## 维度平衡统计（仅正式 17 角色，起点面板≥4 计为"主要维度"）

| 维度 | 作为主要维度的角色数 |
|------|-------------------|
| 志 | 5（赫敏、居里、武则天、Miss Marple、邓亚萍） |
| 幽 | 8（Goodall、李清照、Marple、Frida、Lila、林黛玉、Beth、三毛） |
| 韧 | 5（千寻、李清照、Frida、Isis、邓亚萍） |
| 渡 | 4（赫敏、Grace、Lila、Isis） |
| 身 | 4（Goodall、花木兰、邓亚萍、三毛） |
| 影 | 7（居里、武则天、花木兰、林黛玉、Beth、Elsa、三毛） |

*幽和影仍然偏多，但这符合游戏的内在旅程基调。待开发角色（Zaha、邓亚萍、Onnie Lee 等）上线后会进一步平衡。*
